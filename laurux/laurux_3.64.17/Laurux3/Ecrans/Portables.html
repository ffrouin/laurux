~ Gestion des portables inventaires sous Laurux ~

         

1 ) Généralités.

Si vous disposez d'un portable inventaire, ou d'un téléphone sous Android, il est possible d'utilisez votre matériel avec Laurux pour faire les opérations suivantes :
    Saisie des inventaires de fin d'exercice et tournants, commandes fournisseurs et récéptions de marchandises automatique, factures client, étiquettes produits, mises à jour des codes casier.

Tous les matériels peuvent être utilisés s'ils répondent aux contraintes suivantes :
    En général les matériels sont livrés avec un programme tournant sous Windows. Ce programme devra être lancé sous Linux grâce à Wine. Veuillez vous assurer de cette possibilité avant l'acquisition de votre matériel.
    Le matériel doit être configuré afin de générer un fichier texte nommé "inventaire.txt" .
    Chaque ligne du fichier généré doit avoir la structure suivante : Code barre et quantité séparés par un point virgule.
    Le fichier "inventaire.txt" devra être copié sous un répertoire d'échange nommé "Portable" situé à la racine de l'utilisateur (~/Portable).
 
Pour utiliser votre téléphone sous Android comme un portable inventaire il faudra télécharger l'utilitaire "Lauruxscan" à cette adresse : https://play.google.com/store/search?q=Laurux&c=apps
    Cette application qui a été écrite par Fabrice Lalande fonctionne avec des applications de lecture de code Barre tel que  "Barcode scanner Zxing" ou "Barcode scanner quickmark"



2) Utilisation.

1-Scannez les produits avec l'appareil puis videz celui-ci dans le répertoire ~/Portable.
2- Lancez l'import sur le programme concerné à l'aide du bouton "Importer".
3- Vérifiez le fichier d'anomalies qui s'ouvre en fin d'import.
4- Contrôlez la récéption des données.  

NB : Le fichier "inventaire.txt" n'est pas éffacé par le programme d'import mais (en principe, mais vous devez le vérifiez) est écrasé à chaque nouvelle copie par le portable inventaire. Cela signifie qu'il est possible de faire plusieurs import identiques.
