    Saisies de la Tva sur les ventes

    Positionnez-vous sur la zone " Code " et saisissez le sur 2 caractères maxi, saisissez ensuite le taux, puis le code comptable correspondant.

    Si le taux de la TVA est modifié alors le programme demandera si on souhaite mettre à jour les données de la base (Les produits, les MO et les lignes des documents autres que les factures imprimées.)

    NB : Les historiques des factures ne sont pas impactés par une éventuelle modification de la TVA.

    Important : Faire toujours une sauvegarde avant de faire une modification d'un taux de TVA.

    N’oubliez pas d’enregistrer votre travail avant de quitter le programme.
