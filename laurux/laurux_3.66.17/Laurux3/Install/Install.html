~ Comptabilité et facturation Laurux ~


Installation manuelle.

    Laurux necessite l’installation des packages suivants.

    • MySQL  Base de données (Installer le serveur et le client).
      Outre la base il faut installer les bibliothèques de développement ( Mysql-devel ) si vous compilez Gambas.
    • Gambas3  Langage de programmation
      Généralement livrée avec les principales productions de Linux. Si votre version est trop ancienne ou si vous ne disposez pas de Gambas3 sur votre machine, télécharger la derniere version sur le site de Gambas et après décompression de l’archive suivre la procédure d’installation ( Fichier Install )

      Si vous êtes sous Ubuntu ou dérivé, ous pouvez faire l'installation de Gambas3 en ligne de commande dans une console en utilisant ce ppa :
      sudo add-apt-repository ppa:gambas-team/gambas3

      Puis, il faudra faire :

      sudo apt-get update
      sudo apt-get install gambas3


    • Attention ! Verifiez la présence de gtklp et Okular sous Kde ou bien Evince et Gtklp sous Gnome. Pour l'éditeur utilisé dans certains programmes, on lancera Gedit sous Gnome et Kwrite ou Kate sous Kde. Si vous souhaitez utiliser la facturation périodique (factures abonnements) installez et configurez Exim4 et Mutt car Laurux utilise ces programmes pour les envois groupés (voir la documentation de la facturation périodique).

    • Laurux Programmes et accessoires.
      • Copier et décompacter le fichier téléchargé Laurux3.tar.gz sous votre repertoire.
        Cela va créer un nouveau repertoire nommé Laurux3 qui contiendra les fichiers suivants:
        Laurux3.gambas         (exécutable), Ecrans (Dossier contenant la documentation de laurux), Install (un dossier contenant les instructions d'installation), Laurux01.sql (Base de données ayant une société de travail contenant un plan comptable, des taux de tva sur achat et sur ventes, quelques libéllés, des journaux, des comptes fournisseurs et clients divers, un paramétrage des comptes de résultat et des SIG, les échéances. (Installation facultative mais conseillée lors d'un démarrage).)
        Important : Il faut impérativement décompacter le fichier téléchargé Laurux3.tar.gz sous votre repertoire principal (/home/utilisateur).
    • Lorsque tout est installé, aller dans une console et se logger en root.
      • Lancer MySQL en tapant directement dans une console : sudo mysql -pmotdepasse.
        NB : "mot de passe" est le mot de passe administrateur saisi lors de l'installation de MySQL. Notez qu'il n'y a pas d'espace entre -p et le mot de passe.
      • Créer un utilisateur ayant tous les droits.
        • Ex : GRANT ALL PRIVILEGES ON *.* to votrenomutilisateur@localhost IDENTIFIED BY ’motdepasse’;
          ATTENTION !
          Si vous êtes en configuration réseau mettez l'adresse du host plutôt que l'IP (par exemple toto@paris.local). Si vous faites cela vous pourrez vous connecter aussi bien par le host que par l'IP sinon vous ne pourrez pas vous connecter par le host.
          Le mot de passe doit etre précédé et suivi par des apostrophes, cependant dans certaines productions (ex Ubuntu 10.10) il faut mettre des guillemets à la place des apostrophes.
          N'oubliez pas le point virgule en fin de phrase car une commande SQL se termine toujours par un point virgule.
          Ne mettez pas d'espace ou autres signes comme & \ " dans les noms d'utilisateurs ou dans les mots de passe. Que des lettres ou des chiffres.
      • creer une base. 
      • Pour vérifier votre création:  Sortir de root ( CTRL + D  ou exit ) puis lancer mysql en tapant la commande suivante . 
      • Ex : mysql -h localhost -u votrenomutilisateur -p; Après validation saisir le mot de passe. Si tout est bien crée vous accederez au prompt mysql. 
      • Testez vos droits en créant une base.
        Ex : CREATE DATABASE nomdelabase; Si tout est OK supprimez la base en tapant DROP DATABASE nomdelabase;
    • Verifier éventuellement la présence des fontes Sherif, Century, Bistream et Flubber utilisées par Laurux ( La fonte Flubber est jointe avec les fichiers 'Install' téléchargés ). Pour installer une nouvelle fonte, faire un clic droit sur son icone puis cliquer sur "Action" et "installer". Créer enfin une icône sur votre bureau qui chainera sur l'executable Laurux3.gambas situé sous le répertoire ~/Laurux3. Vous pouvez associer l'image logo.png située sous le meme répertoire .

Première utilisation.

       Le nom de la base devra être composé d'une racine, Laurux, puis d'un numéro de société, 01, 02, 03 etc...
       Lorsque vous lancerez Laurux pour la première fois, vous arriverez sur le menu de création de la base de données et des préférences. La base qui sera créée s'appelera impérativement Laurux01 parce que le programme, après la création de la base, effectuera lui-même une restauration d'une sauvegarde nommée Laurux01.sql située sous ~/Laurux3



       Saissez le nom d’utilisateur MySQL ainsi que le mot de passe tels que vous les avez crées sous MySQL.
       Saissez l’adresse du serveur ainsi que le port. Si vous n’étes pas sûr de ces données, laissez donc ce qui est proposé par défaut. Ne remplissez pas les zones SSH tout de suite. Cela se fera ultérieurement par le programme de connexion de base.
       Dans le cas d'une utilisation multi-utilisateurs, il faudra définir le poste sur lequel se feront les mises à jour de la base et activer sur ce dernier le bouton "A cocher si c'est le cas". En monoposte ce bouton devra être toujours être coché.
       Cliquez sur le bouton "Valider". Si tout est correctement parametré sous MySQL, la base de démarrage nommée Laurux01 sera créée et restaurée. Un message vous avertira du succès de l’opération. En cas de souci vérifiez les droits, sous MySQL, de l’utilisateur qui doit fonctionner par défaut avec Laurux.
       Commencez par aller dans la table des sociétés afin de rentrer vos données personnelles puis allez dans les paramètres et adaptez les dates des périodes à votre convenance. Ensuite allez dans les préférences afin d'adapter votre environnement de travail à votre convenance ou vos besoins.
       Voilà, à ce stade Laurux est opérationnel et vous pouvez commencer votre comptabilité et votre facturation.

        NB: Si vous arrivez sans arret sur le menu de configuration en lancant Laurux, essayer de supprimer le fichier caché ~/.config/gambas3/laurux3.conf puis relancez Laurux. Vérifiez que le répertoire Laurux3 soit bien situé sous votre répertoire personnel.


Note sur l'utilisation de QTconfig.

Laurux3 utilise la bibliothèque graphique QT. Pour améliorer l'apparence de Laurux3, il est nécessaire de procéder à quelques réglages de QT.
Comment faire ?
1- Installer Qt4-qtconfig à l'aide d'un gestionnaire de paquets tel que Synaptic.
2- Aller dans les préférences et lancer le programme "Paramètres QT4"
3- Dans la zone Gui-style, séléctionner le style "Plastique".
4- Allez dans le "Tune palette" et mettez les couleurs des Tooltypebase et Tooltypetext selon votre goût.
5- Sortir et sauvegarder.

NB : La résolution optimale est de 1280*1024. Avec des résolutions d'écrans inférieures il peut y avoir des problèmes d'affichage, bien qu'on puisse jouer avec les tailles et les résolutions des polices systèmes utilisées.
