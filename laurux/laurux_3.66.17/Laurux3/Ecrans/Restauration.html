~ Comptabilité Laurux ~


 

    Restauration de la base



    Ce programme va restaurer sur la base Laurux les données du fichier de sauvegarde nommé Laurux.sql . Ce fichier doit donc exister sous le repertoire renseigné dans la zone "support de sauvegarde", par défaut c'est le chemin défini dans les préferences qui apparait mais ce chemin est modifiable.
    NB : Si la base a été compréssée lors de la sauvegarde alors le nom du fichier sera Laurux.sql.gz 

    Rappel : Ayez plusieurs supports de sauvegardes et utilisez-les en alternance.
