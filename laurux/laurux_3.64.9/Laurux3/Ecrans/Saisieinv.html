    Saisie de l'inventaire

    L'impression effectuée apres la génération de l'inventaire se présente selon la copie d'écran ci-dessus.Remarquer que chaque ligne est numérotée
    Vous allez donc noter sur chaque ligne de produit la quantité physique relevée puis saisir ces quantités sur votre informatique.
    Cliquer sur le bouton "Traitement" pour lancer la génération de l'inventaire, puis cliquer sur le bouton "Imprimer" pour lancer l'impression de votre inventaire qui vous servira pour la saisie.

    Dans le programme de saisie, taper en premier le numéro de ligne correspondant a votre saisie puis apres validation saisir la quantité notée sur votre feuille. Valider votre saisie, vour remarquerez que le numero de ligne s'incrémente automatiquement et que vous n'avez besoin de ne saisir que la quantité.

          Si vous possedez un portable inventaire ou un Téléphone sous Android, il est possible de récupérer les données saisies sur ces appareils en cliquant sur le bouton "Importer".
          En import, plusieurs cas sont possibles.
          1- Si le produit n'a pas été saisi alors il est intégré dans l'inventaire.
          2- Si le produit a déjà été saisi alors il est rejeté et une ligne est écrite dans le fichier de contrôle avec la mention "Déjà saisit". Dans ce cas, il faudra faire une saisie manuelle qui, elle, gère ça très bien.
          3- Si le produit n'existe pas dans l'inventaire alors une ligne est écrite dans le fichier de contrôle avec la mention "Code produit absent de l'inventaire ou code barre inexistant"
    Cliquer sur le bouton "Quitter" pour sortir du programme. Vous pouvez sortir du programme à n'importe quel moment puisque vous retrouverez l'ensemble des produits inventoriés à chaque fois que vous reviendrez dans la saisie de l'inventaire.
