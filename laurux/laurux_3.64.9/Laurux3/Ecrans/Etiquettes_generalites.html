~ Comptabilité et Facturation Laurux ~


 

        Principe de la gestion de stock sous Laurux.

            Laurux peut imprimer les étiquettes sur trois types d'imprimantes différentes. Le type d'imprimante utilisé se déclare dans les préférences (onglet 3).
            Imprimante A4 classique, laser ou jet d'encre. Laurux peut utiliser votre imprimante A4, locale ou réseau, pour imprimer les étiquettes.
            Imprimante étiquettes Zebra. Les imprimantes Zebra sont des imprimantes réseau et leur adresse se déclare dans les préférences. Aucun drivers n'est nécessaire. L'installation est ardue pour celui qui ne maitrise pas le réseau. Le paramètrage du fichier d'impression est complexe.
            Imprimante étiquettes Dymo. les imprimantes étiquettes Dymo s'utilisent comme une imprimante A4 normale. Un driver pour GNU/Linux est livré avec le produit et est aussi disponible sur le site Dymo. Par exemple pour une Dymo labelwriter 450, le modèle qui a été utilisé pour les tests, le driver est ici. http://global.dymo.com/frCH/Software/LabelWriter_450.html . L'installation et le paramètrage sont simples et semblabes à ceux des imprimantes A4.

            Laurux propose la gestion de quatre types d'étiquettes. Produits, casiers, gondoles et clients.

            Pour commencer il va falloir paramétrer chacune de ces catégories d'étiquettes par le menu "Types étiquettes" situé dans les constantes de la facturation. Voir la page Paramétrage des étiquettes.

            Ensuite il faudra aller dans les fiches produits pour cocher le bouton "Génération étiquettes" situé dans l'onglet stock si vous souhaitez avoir des étiquettes. 

    1- Les impressions d' étiquettes à la demande.

            Les impressions des étiquettes produits, casiers et gondoles. Ces menus sont accessibles soit par les options "Editions des tables " ==> "Etiquettes", soit par le menu "Stock" ==> "Etiquettes".  Voir la page  Etiquettes produits.

            On ne pourra imprimer des étiquettes produits que si le bouton Had hoc de la fiche article est activé et que si le produit possède un code barre.  
            En revanche aucun contrôle n'est fait pour les étiquettes casiers et gondoles. 

    2- Les étiquettes de réception de marchandises.

            Les impressions des étiquettes réception. Ce menu est accessible par le menu "Stock" ==> "Etiquettes" ==> "Etiquettes réception".  

            Les étiquettes produits et casiers seront générées lors d'une réception de marchandises seulement si le bouton idione de la fiche article est activé et seulement si le produit possède un code barre. 
            Par contre, les étiquettes de gondole seront automatiquement générées suite à une modification de prix lors d'une réception, même si le bouton n'est pas coché.       
            A partir de ce menu vous pouvez imprimer ou effacer les étiquettes produits ou de gondole préparées lors des réceptions manuelles ou automatiques. Voir la page Etiquettes réception

    3- Les étiquettes clients.    

            Les impressions des étiquettes clients. Elles sont accessibles par le menu "Editions des tables" ==> "Etiquettes".  Voir la page Etiquettes client.
