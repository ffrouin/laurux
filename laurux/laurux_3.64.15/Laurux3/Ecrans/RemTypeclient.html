~ Comptabilité et Facturation Laurux ~


 

Table des remises par type client.


Cet écran vous permet de saisir les remises pour chacun des types clients créés dans la table des types client ( Constantes ).
1- Saisir le type du client.
2- Saisir le code la famille utilisée pour la remise. 
3- Saisir le montant de la remise. Vous pouvez utiliser 3 décimales.

Par la suite,  si un produit d'une famille présente dans cette table est saisit en facturation alors la remise de la table sera appliquée automatiquement.
