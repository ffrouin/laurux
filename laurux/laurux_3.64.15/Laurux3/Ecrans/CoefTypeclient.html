~ Comptabilité et Facturation Laurux ~


 

Table des coefficients de vente par type client.


Cet écran vous permet de saisir des coefficients de ventes pour chacun des types clients créés dans la table des types client ( Constantes ).
1- Saisir le type du client.
2- Saisir le code la famille utilisée pour le coerfficient. 
3- Saisir le montant du coefficient. Vous pouvez utiliser 3 décimales.

Par la suite, si un produit d'une famille présente dans cette table est saisit en facturation alors c'est le coefficient de cette table qui sera utilisé au lieu de celui de la fiche produit.

Attention ! Si des remises automatiques sont programmées pour le même produit (par exemple remise client par famille), alors elles s'appliqueront. Donc prudence.
