    Etiquettes de réception.

           Ce programme qui vous permet d'imprimer les étiquettes produits ou de gondole générées par une réception de marchandises se trouve dans le menu Stock ==> Etiquettes.


        Vous avez la possibilité d'imprimer soit les étiquettes produits soit les étiquettes casiers soit les étiquettes de gondole.
        Sous chaque type d'étiquette apparait le nombre d'étiquettes qui sera imprimé.

        On peut fournir un numéro d'ordre pour le début de l'impression, ce qui est utile lorsqu'une planche d'étiquettes A4 a été en partie déjà utilisée.

        Vous pouvez aussi effacer, pour chaque type d'étiquettes, les étiquettes en cours en cliquant sur le bouton "Raz fichier".
